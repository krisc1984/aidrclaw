# React Native WebRTC 集成参考
<interfaces>
From react-native-webrtc documentation:
```typescript
import { mediaDevices, MediaStream } from 'react-native-webrtc';

// 获取摄像头权限
const stream = await mediaDevices.getUserMedia({
  video: { facingMode: 'user' },
  audio: true
});

// 切换前后摄像头
const newStream = await mediaDevices.getUserMedia({
  video: { facingMode: facing === 'user' ? 'environment' : 'user' }
});
```

       - 使用 PermissionsAndroid (Android) / check/request (iOS) 请求权限

    2. 实现摄像头切换逻辑：

    3. 处理 App 生命周期：
       - App 切到后台时暂停录制（保存当前状态）
       - App 返回前台时恢复录制
    4. 错误处理：
       - 权限拒绝：显示"需要在设置中开启摄像头权限"
       - 设备不可用：显示"未找到可用摄像头"
  </action>
