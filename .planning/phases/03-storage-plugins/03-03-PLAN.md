    2. **默认实现** (DefaultStorageStrategy):
       ```java
       @Component
       public class DefaultStorageStrategy implements StorageStrategy {
           @Value("${storage.default-backend:local}")
           private String defaultBackend;
           
           @Value("${storage.large-file-threshold:104857600}") // 100MB
           private long largeFileThreshold;
           
           @Override
           public String selectBackend(long fileSize, String businessType, boolean encrypted) {
               // 规则 1: 加密文件优先使用 MinIO
               if (encrypted) {
                   return "minio";
               }
               
               // 规则 2: 临时文件使用本地存储
               if ("temporary".equals(businessType)) {
                   return "local";
               }
               
               // 规则 3: 大文件使用 MinIO
               if (fileSize >= largeFileThreshold) {
                   return "minio";
               }
               
               // 默认
               return defaultBackend;
           }
           
           @Override
           public int getRetentionDays(String businessType) {
               // 可根据业务类型配置不同的保留天数
               return 365; // 默认 1 年
           }
       }
       ```

  <action>
    实现 StorageOrchestrator 统一管理存储和加密流程：

    1. **核心职责**:
       - 接收文件保存请求
       - 调用 StorageStrategy 选择存储后端
       - 根据配置决定是否加密
       - 调用对应的存储插件保存
       - 元数据入库

    2. **实现类**:
       ```java
       @Component
       public class StorageOrchestrator {
           @Autowired
           private PluginManager pluginManager;
           
           @Autowired
           private StorageStrategy storageStrategy;
           
           @Autowired(required = false)
           private EncryptionArchivePlugin encryptionPlugin; // 可选
           
           @Autowired
           private StorageFileMapper storageFileMapper;
           
           /**
            * 保存文件（自动选择存储和加密）
            */
           public StorageFile saveFile(InputStream inputStream, String businessType, 
                                       String businessId, boolean requireEncryption) {
               // 1. 读取文件并计算哈希
               byte[] data = IOUtils.toByteArray(inputStream);
               String hash = Hashing.sha256().hashBytes(data).toString();
               
               // 2. 决定是否加密
               boolean encrypted = requireEncryption && encryptionPlugin != null;
               
               // 3. 选择存储后端
               String backend = storageStrategy.selectBackend(data.length, businessType, encrypted);
               
               // 4. 获取对应的存储插件
               Plugin storagePlugin = pluginManager.getPlugin(
                   "minio-storage".equals(backend) ? "minio-storage" : "local-storage"
               );
               
               // 5. 如果需要加密，调用加密插件
               PluginResult saveResult;
               if (encrypted) {
                   PluginContext encryptContext = new PluginContext();
                   encryptContext.getInput().put("action", "saveEncrypted");
                   encryptContext.getInput().put("data", data);
                   encryptContext.getInput().put("businessType", businessType);
                   encryptContext.getInput().put("businessId", businessId);
                   saveResult = encryptionPlugin.execute(encryptContext);
               } else {
                   PluginContext saveContext = new PluginContext();
                   saveContext.getInput().put("action", "save");
                   saveContext.getInput().put("inputStream", new ByteArrayInputStream(data));
                   saveContext.getInput().put("businessType", businessType);
                   saveContext.getInput().put("businessId", businessId);
                   saveResult = storagePlugin.execute(saveContext);
               }
               
               // 6. 元数据入库
               StorageFile storageFile = new StorageFile();
               storageFile.setFilePath(saveResult.getData().get("filePath").toString());
               storageFile.setFileSize((long) data.length);
               storageFile.setFileHash(hash);
               storageFile.setBusinessType(businessType);
               storageFile.setBusinessId(businessId);
               storageFile.setStorageBackend(backend);
               storageFile.setEncrypted(encrypted);
               // ... 设置其他字段 ...
               
               storageFileMapper.insert(storageFile);
               
               return storageFile;
           }
       }
       ```

    3. **使用示例**:
       ```java
       // 在 UploadController 中调用
       @PostMapping("/upload")
       public ResponseEntity<?> upload(@RequestParam("file") MultipartFile file) {

               sessionId,
               true // 要求加密
           );
           return ResponseEntity.ok(saved);
       }
       ```
