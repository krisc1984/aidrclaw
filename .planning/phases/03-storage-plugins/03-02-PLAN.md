<task type="auto">
  <name>Task 3: 加密归档插件实现</name>
  <files>aidrclaw-plugins-storage/EncryptionArchivePlugin.java</files>
  <action>
    实现 EncryptionArchivePlugin，整合加密和存储能力：

    1. **插件元数据**:
       - pluginId: "encryption-archive"
       - version: "1.0.0"
       - name: "加密归档插件"
       - description: "AES-256-GCM 加密存储"

    2. **依赖注入**:
       - @Autowired EncryptionKeyStore keyStore
       - @Autowired PluginManager pluginManager (获取存储插件)

    3. **支持的 action**:
       - "saveEncrypted": 加密后保存
         - 输入：{ inputStream, businessType, businessId }
         - 处理流程:
           1. 读取输入流到字节数组
           2. 计算 SHA-256 哈希（加密前）
           3. 使用 EncryptionUtils.encrypt() 加密
           4. 调用存储插件的 "save" action 保存密文
           5. 元数据记录：{ encrypted: true, keyId, iv, hash }
         - 输出：{ fileId, filePath, encryptionMetadata }

       - "loadDecrypted": 解密后读取
         - 输入：{ fileId }
         - 处理流程:
           1. 从数据库查询加密元数据
           2. 调用存储插件的 "load" action 获取密文
           3. 使用 EncryptionUtils.decrypt() 解密
           4. 计算 SHA-256 哈希（解密后）并校验
         - 输出：{ inputStream, originalHash }

    4. **错误处理**:
       - 加密失败 → PluginResult.error("ENCRYPTION_FAILED", message)
       - 解密失败 → PluginResult.error("DECRYPTION_FAILED", message)
       - 密钥不存在 → PluginResult.error("KEY_NOT_FOUND", keyId)

    5. **日志**:
       - 记录加密操作（不含密钥信息）
       - 记录文件大小、耗时
  </action>
