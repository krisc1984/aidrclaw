创建 Spring 配置类，配置 SensitiveWordBs Bean：

```java
@Configuration
public class SensitiveWordConfig {
    
    @Bean
    public SensitiveWordBs sensitiveWordBs() {
        return SensitiveWordBs.newInstance()
            .ignoreCase(true)              // 忽略大小写
            .ignoreWidth(true)             // 忽略半角圆角
            .wordFailFast(true)            // 快速匹配模式
            .wordDeny(WordDenys.chains(    // 组合词库
                WordDenys.defaults(),      // 默认词库
                new FinancialViolationWords()  // 金融违规词库
            ))
            .wordAllow(WordAllows.chains(  // 白名单
                WordAllows.defaults()      // 默认白名单
            ))
            .init();
    }
}
```

配置说明：
- ignoreCase: true - 忽略大小写（中文不受影响）
- ignoreWidth: true - 忽略半角圆角
- wordFailFast: true - 快速匹配模式（性能优先）
- wordDeny: 默认词库 + 金融违规词库组合
- wordAllow: 默认白名单（避免误报）

创建敏感词检测服务：

```java
@Service
public class SensitiveWordService {
    
    @Autowired
    private SensitiveWordBs sensitiveWordBs;
    
    /**
     * 检测文本中的违规词
     * @param transcript 对话文本
     * @return 违规词列表
     */
    public List<String> detectViolations(String transcript) {
        return sensitiveWordBs.findAll(transcript);
    }
    
    /**
     * 检查是否包含违规词
     * @param transcript 对话文本
     * @return true-包含违规词
     */
    public boolean containsViolation(String transcript) {
        return sensitiveWordBs.contains(transcript);
    }
    
    /**
     * 动态添加违规词（支持热更新）
     * @param words 违规词列表
     */
    public void addWords(List<String> words) {
        words.forEach(word -> sensitiveWordBs.addWord(word));
    }
}
```
