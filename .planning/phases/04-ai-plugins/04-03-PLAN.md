  <action>
    Create TTS plugin that implements Plugin interface:
    
    1. Create TtsConfig.java:
       - @ConfigurationProperties(prefix = "ai.tts")
       - Fields: enabled (boolean), url (String), timeout (int), defaultSpeaker (int), defaultSpeed (float)
       - Default url: http://localhost:8001
    
    2. Create TtsService.java:
       - @Service annotation
       - Inject RestTemplate or HttpClient
       - Method: synthesize(String text, int speakerId, float speed) → byte[]
         * HTTP POST to {url}/tts/synthesize with JSON body
         * Content-Type: application/json
         * Response: audio/wav binary
         * Extract duration from X-Audio-Duration header
         * Return audio bytes
       - Method: getVoices() → List<VoiceInfo>
         * HTTP GET to {url}/tts/voices
         * Parse JSON to VoiceInfo list
       - Handle timeout, errors
    
    3. Create TtsPlugin.java:
       - @Component annotation
       - Implement Plugin interface
       - @Autowired TtsService and TtsConfig
       - execute() method:
         * Extract "text" from PluginContext.getInput()
         * Extract optional "speakerId" (default from config)
         * Extract optional "speed" (default 1.0)
         * Call ttsService.synthesize(text, speakerId, speed)
         * Put "audio", "duration" into PluginResult
         * Handle errors: return PluginResult.error(500, message)
       - getMetadata(): return name="TTS Plugin", version="1.0.0", description="语音合成插件"
  </action>
  <behavior>

    - Test: execute() with different speakerId → different audio
    - Test: execute() with empty text → returns error 400
    - Test: execute() when service unavailable → returns error after timeout
    - Test: metadata contains correct name and version
  </behavior>
