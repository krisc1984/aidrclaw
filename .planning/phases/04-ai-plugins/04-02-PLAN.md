  <action>
    Create face recognition plugin that implements Plugin interface:
    
    1. Create FaceConfig.java:
       - @ConfigurationProperties(prefix = "ai.face")
       - Fields: enabled, url, timeout, matchThreshold (default 0.60), livenessThreshold (default 0.50)
    
    2. Create DTOs in src/main/java/com/aidrclaw/ai/face/dto/:
       - FaceCompareRequest.java (image1, image2 as byte[])
       - FaceCompareResponse.java (similarity, match, bbox1, bbox2)
       - LivenessRequest.java (image as byte[])
       - LivenessResponse.java (liveness, score, bbox)
    
    3. Create FaceService.java:
       - @Service annotation
       - Inject RestTemplate or HttpClient
       - Method: compare(byte[] image1, byte[] image2) → FaceCompareResponse
         * HTTP POST to {url}/face/compare with multipart images
         * Parse JSON response to DTO
       - Method: detectLiveness(byte[] image) → LivenessResponse
         * HTTP POST to {url}/face/liveness
         * Parse JSON response
       - Handle errors: no face, timeout, service unavailable
    
    4. Create FacePlugin.java:
       - @Component annotation
       - Implement Plugin interface
       - @Autowired FaceService and FaceConfig
       - execute() method:
         * Extract "action" from context: "compare" or "liveness"
         * If "compare": extract image1, image2 → call faceService.compare()
           - Put "similarity", "match", "message" into result
         * If "liveness": extract image → call faceService.detectLiveness()
           - Put "liveness", "score", "message" into result
         * Handle errors: return PluginResult.error(statusCode, message)
       - getMetadata(): return name="Face Recognition Plugin", version="1.0.0"
  </action>
  <behavior>
    - Test: execute() with action="compare" → returns similarity and match
    - Test: execute() with action="liveness" → returns liveness and score
    - Test: execute() with unknown action → returns error 400
    - Test: compare with high similarity (>0.60) → match=true
    - Test: liveness with real face → liveness=true
  </behavior>
