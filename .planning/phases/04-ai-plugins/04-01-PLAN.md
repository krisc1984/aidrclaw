  <action>
    Create ASR plugin that implements Plugin interface:
    
    1. Create AsrConfig.java:
       - @ConfigurationProperties(prefix = "ai.asr")
       - Fields: enabled (boolean), url (String), timeout (int), retry (int)
       - Default url: http://localhost:8000
    
    2. Create AsrService.java:
       - @Service annotation
       - Inject RestTemplate or HttpClient
       - Method: recognize(byte[] audio) → AsrResult {text, confidence, duration}
       - HTTP POST to {url}/asr/recognize with audio/wav content-type
       - Handle timeout, retry logic (max 3 retries)
       - Convert JSON response to AsrResult
    
    3. Create AsrPlugin.java:
       - @Component annotation
       - Implement Plugin interface
       - @Autowired AsrService and AsrConfig
       - execute() method:
         * Extract "audioData" from PluginContext.getInput()
         * Call asrService.recognize(audioBytes)
         * Put "text", "confidence", "duration" into PluginResult
         * Handle errors: return PluginResult.error(500, message)
       - getMetadata(): return name="ASR Plugin", version="1.0.0", description="语音识别插件"
  </action>
  <behavior>
    - Test: execute() with audio bytes → returns PluginResult with text field
    - Test: execute() with null audio → returns error with status 400
    - Test: execute() when service unavailable → returns error after 3 retries
    - Test: metadata contains correct name and version
  </behavior>
